## YOMI

## Tested
OS: MACOS X, Linux

CL implementation: SBCL, CCL


## Parameters
\*keymap\* : "emacs", "vim" or "sublime"

\*max-eval-threads\* : maximum number of evaluation threads

## Keyboard Shortcut
Ctrl-Return : Evaluate focused cell

Ctrl-s : Save notebook


## Quick start
\(yomi:start-yomi\)
